# 🚀 CodePair Studio

**The All-in-One Collaborative Development Suite** *Empowering remote engineering teams with real-time code execution, AI assistance, and multimodal communication.*

---

## 🌟 Overview

CodePair Studio is a high-performance, real-time collaborative IDE designed to eliminate context-switching. Built for developers by developers, it integrates a sophisticated code editor with a whiteboard, audio/video conferencing, and an AI-powered copilot into a single, native desktop environment.

## ✨ Key Features

### 💻 Core IDE & Execution
* **Real-time Synchronization:** Sub-100ms latency for collaborative editing powered by **Socket.io**.
* **Multi-Language Support:** Execute code in **88+ languages** instantly via the **JDoodle API**.
* **Virtual File System:** Full CRUD operations on project files with the ability to **Download as ZIP** or upload entire codebases.
* **Admin Controls:** Granular room management including edit permissions, participant kicking, and presence indicators.

### 🤖 AI-Powered Collaboration
* **AI Copilot:** Context-aware code generation and debugging powered by **Google Gemini API**.
* **Group Chat:** Integrated real-time messaging for rapid-fire technical discussions.

### 📞 Multi-Modal Communication
* **Audio & Video Calling:** High-fidelity, low-latency WebRTC conferencing using the **Stream Video/Audio SDK**.
* **Integrated Whiteboard:** A multiplayer infinite canvas using **tldraw** for architecture diagramming and visual brainstorming.

### 🖥️ Native Desktop Experience
* **Cross-Platform:** Packaged with **Electron.js** for Windows, macOS, and Linux.
* **Native Performance:** Access to OS-level shortcuts and optimized resource management via **electron-builder**.
